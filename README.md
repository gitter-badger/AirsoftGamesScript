# サバゲ主催者用スクリプト集

## 趣旨

* 貸切ゲームでの朝ブリーフィングの説明事項の漏れをなくす
* 説明漏れや、適切な強調を行うことで事故をなくす
* 主催者の負荷軽減
* 主催のハードル低下
